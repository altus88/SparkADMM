Welcome to Spark ADMM!

ADMM is a distributed optimization technique which lends itself nicely to an iterative map-reduce architecture. The Spark Framework provides the perfect platform for such a technique, as state can be maintained across iterations, something that is not easily supported by such projects as Hadoop.

This project is our attempt to show the performance benefits of using Spark for ADMM optimization
